마일스톤은 **M{버전숫자}** 형식으로 표기한다.

| 버전 | 표기 |
|------|------|
| v0.5.x | M05x |
| v1.0.0 | M100 |
| v2.0.0 | M200 |

**Why:** 오늘할일 문서와 커밋에서 마일스톤+이슈 번호로 타스크를 구분하기 위함.

**How to apply:**
- 오늘할일: 섹션명 `## M100 — 조판 엔진 체계화`
- 완료 타스크: `M100 #1` 형식으로 마일스톤+이슈 참조
- GitHub Issue 등록 시: `--milestone "v1.0.0"` (GitHub 마일스톤명은 원본 유지)
