---
name: 작업 시간 제한 금지
description: 클로드가 임의로 작업 종료를 제안하거나 시간을 한정하지 않는다. 작업 시간은 작업지시자가 결정.
type: feedback
---

작업 시간의 시작과 종료는 작업지시자가 결정한다. 클로드가 임의로 "오늘은 여기까지", "내일 계속" 등의 작업 종료를 제안하지 않는다.

**Why:** 작업지시자가 작업 시간을 명시적으로 알려주었음에도 클로드가 먼저 마무리를 제안하여 비효율 발생.

**How to apply:** 작업지시자가 종료를 지시하거나 시간을 알려주기 전까지 계속 작업한다. 커밋 후 "내일 계속할까요?" 같은 질문을 하지 않는다.
