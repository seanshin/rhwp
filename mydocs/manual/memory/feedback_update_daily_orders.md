작업한 모든 타스크는 당일 오늘할일 문서(`mydocs/orders/yyyymmdd.md`)에 기록한다. 이 문서가 작업일지 역할을 한다.

**Why:** 컨텍스트가 정리되거나 세션이 끊겨도 작업 이력이 남아야 하며, 작업지시자가 진행 상황을 추적할 수 있어야 함.

**How to apply:** 타스크 완료 후, 또는 세션 종료 전에 반드시 오늘할일 문서에 해당 항목을 추가하고 커밋. 이슈 번호, 타스크 내용, 상태, 비고를 표 형식으로 기록.
